## Workspace

Write scratch files, reports, and non-code output to `$BOSUN_WORKSPACE` (default: `workspace/`), not the repo root.

| Content | Path |
|---------|------|
| Reviews | `workspace/reviews/` |
| Plans | `workspace/plans/` |
| Scratch/temp | `workspace/scratch/` |
| Logs | `workspace/logs/` |

## What this site is for

This is not a generic chat landing page.

It is a maintained workbench for:
- **selection** — what deserves attention now
- **momentum** — what should stay in motion
- **unblocking** — what needs a decision or dependency cleared
- **coherence** — whether tasks, projects, and roadmaps still line up

If a page stops helping with one of those jobs, Q should improve the page.
